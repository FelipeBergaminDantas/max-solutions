"Max Solutions & Performance"

O npm (gerenciador de pacotes do Node.js) NÃO FUNCIONA com espaços no 
caminho no Windows. Este é um problema conhecido e documentado.

================================================================================
✅ SOLUÇÃO SIMPLES (3 passos)
================================================================================

1. MOVA o projeto para um diretório SEM ESPAÇOS
   Exemplo: C:\projetos\max-solutions

2. INSTALE as dependências:
   npm install

3. RODE o projeto:
   npm run dev

4. ACESSE no navegador:
   http://localhost:3000
