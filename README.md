# Max Solutions & Performance

Site institucional da Max Solutions & Performance - Consultoria Certificada Mercado Livre.

## 🚀 Tecnologias

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide Icons

## 📦 Instalação

```bash
npm install
```

## 🔧 Desenvolvimento

```bash
npm run dev
```

Abra [http://localhost:3000](http://localhost:3000) no navegador.

## 🏗️ Build

```bash
npm run build
npm start
```

## 📝 Estrutura do Projeto

```
├── app/
│   ├── layout.tsx       # Layout principal
│   ├── page.tsx         # Página inicial
│   └── globals.css      # Estilos globais
├── components/
│   ├── Header.tsx       # Cabeçalho
│   ├── Hero.tsx         # Seção hero
│   ├── About.tsx        # Sobre nós
│   ├── Clients.tsx      # Casos de sucesso
│   ├── Contact.tsx      # Contato
│   └── Footer.tsx       # Rodapé
└── public/              # Arquivos estáticos
```
